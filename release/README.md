# Portable Package

This directory contains a ready-to-download archive for the current customized `cac` repo.

Files:

- `cac-portable-latest.zip`
- `cac-portable-latest.sha256`

Usage:

1. Download `cac-portable-latest.zip` from GitHub.
2. Unzip it anywhere.
3. Enter the extracted directory.
4. Run `bash install.sh`.

The archive keeps the original project files intact and adds no extra runtime state such as `.git`, `.cac`, or `.cac-dist`.
